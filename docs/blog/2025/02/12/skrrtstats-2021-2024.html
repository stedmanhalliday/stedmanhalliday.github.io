Generate your own insights

Generate insights for any artist of your choice from your own Spotify streaming data. Here’s how:

  1. Visit the Spotify Account privacy page and log in
  2. Request a copy of your personal data with the Download your data tool
    • Use the normal option, not Extended streaming history
    • It will take some time for Spotify to process your data and email you back with a download link
    • Download and unarchive your data
  3. Download code from this gist, or copy it to a new file from the block below
  4. Place the script file in the same directory as the unarchived data folder (Spotify Account Data)
  5. Edit the constants up top for the artist and number of top tracks you’d like to output
  6. Run the script from the command line with Node.js or an equivalent JavaScript runtime environment
  • Bonus: use Carbon to generate cool snapshots of your stats like the ones I’ve featured in this post
