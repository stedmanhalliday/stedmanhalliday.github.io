Skip/Crush is an eight-channel projection-mapped video installation set to the track "Skip/Crush" by Canadian techno duo Graze. I programmed generative, audio-responsive graphics and mapped them across the space in Quartz Composer, Ableton Live, and Resolume Arena.

The installation can run from a MIDI score or be performed live with MIDI input, so the visuals are an instrument as much as a fixed piece. See the [live performance](https://www.youtube.com/watch?v=BmXhTKkAY_c).
